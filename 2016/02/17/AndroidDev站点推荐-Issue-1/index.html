AndroidDev站点推荐-Issue#1

2016-02-17
  • Android-Dev-Bookmarks

Android-Dev-Bookmarks 项目目前可以生成书签文件导入到浏览器中了，ADB 站点上的内容也是一样的，方便在线浏览。现在内容更新开始按期发布同时也会有个简短的介绍，内容逐渐增多比较难直接看出站点的主要内容，过一遍介绍也可以留个印象。

下面是这期更新的内容：

ADB-Issue#1

ADB 站点主页：adb.rocko.xyz
GitHub 项目主页： Android-Dev-Bookmarks
点我订阅每期更新
点我下载书签文件导入浏览器

官方资料

博客

  • Google testing blog - Google 测试团队博客，发布一些 Google Testing 的新闻和更新。

文档

  • Google Java Style - Google Java 编码规范文档
  • High-Performance Audio on Android - 创建高性能的音频应用，一些开发向导、相关资料链接、代码示例。

代码-库

开源 App

  • Open-Source Android Apps - GitHub 上的项目，收集优秀的开源 App，分类详细内容多。

各种库

  • libtastic - 通过流行的应用来发现 Top Android libraries，支持应用搜索。
  • AppBrain - AppBrain 上分析的库使用排行。

服务平台

测试

  • TestBird - TestBird APP和手游测试专家是全球第一手游自动化云测试和移动APP测试平台,以高效手游及APP测试流程和16000多款手机游戏/APP测试用例完成超过1万份测试报告，为全球5500多家手机游戏/移动应用厂家和渠道提供手游APP测试服务。
  • 优测网 - 腾讯优测是备受客户信赖的移动云测试平台，为应用、游戏、H5混合应用的研发团队提供产品质量检测与问题解决服务。不仅在线上平台提供全面兼容测试、源码缺陷分析、远程真机租用等多种质量检测工具，更为VIP客户配备了专家团队提供定制化综合测试解决方案。

应用

发行

  • fastlane - 一系列方便移动开发发布应用的工具，大大提高效率节省时间。

API

  • SHOWAPI-易源接口 - 易源是一个统一了通讯协议的API总线，互联网接口调用的入口，高度安全，超高性能，海量接口资源。一次接入，全部可调。可自由定制返回数据结构。角色：API提供者，发布并监控API，设置调用权限和调用费率；API调用者，使用统一协议格式高速调用海量API资源池。

其它

  • buddybuild - 类自动集成，登录账号可以直接构建 GitHub、Bitbucket、Gitlab 等仓库上的项目。

博客

Android

国外

  • Android Bookmarks - Android文章分类汇总，方便的分类查找、学习，英文的。

其它

  • 500px Engineering Blog - 500PX 团队博客

工具-资源

综合

  • AppBrain Stats - Android Operating System Statistics，包括 Play 上的 App 的各种状态、Android 设备的各种状态、Android library 使用的各种状态和排行等等。

UI 界面设计

资源工具

  • Facebook Design Resources - Facebook 设计师设计的免费设计资源。
  • UpLabs - 为开发者和设计师而生，每日发现新鲜的设计和开发资源、寻找一个特定的用户界面解决方案或库。
  • Android™ Drawables - 发现和比较不同版本 Android 平台上的资源文件。

文档编写

  • ASCIIFlow Infinity - 在线画 ASCII 码字符图表，支持导入导出和保存到 Google Drive。

其它

  • Snapdrop - 通过 Snapdrop 网站方便地在设备间传送文件，两台设备同时打开此网站即可操作。

活动会议

  • MDay - Mobile Developers Day - MDay 是蘑菇街举办的面向移动开发者的分享交流会，此交流会会邀请一些一线的资深开发工程师来分享日常开发中遇到的各种问题、经验总结、解决思路、最佳实践等。
