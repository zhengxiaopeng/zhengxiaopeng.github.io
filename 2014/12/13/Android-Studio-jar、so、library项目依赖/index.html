Android Studio JarгҖҒsoгҖҒlibraryйЎ№зӣ®дҫқиө–

2014-12-13
  • Android

еүҚиЁҖ

Android Studio(д»ҘдёӢз®Җз§°AS)еңЁ13е№ҙI/OеӨ§дјҡеҗҺж”ҫеҮәйў„и§ҲзүҲеҲ°зҺ°еңЁж”ҫеҮәзҡ„жӯЈејҸзүҲ1.0пјҲPS.д»ҠеӨ©еҸҲжӣҙж–°еҲ°1.0.1дәҶпјүеҺҶж—¶дёҖе№ҙеӨҡдәҶпјҢиҷҪз„¶Googleе®ҳж–№жҺЁеҮәзҡ„AndroidејҖеҸ‘иҖ…зҡ„IDEеҜ№жҲ‘д»¬Android DEVжҳҜеҫҲжңүеҗёеј•еҠӣзҡ„пјҢдҪҶиҖғиҷ‘еҲ°betaзүҲиҝҳжҳҜеӨӘеӨҡй—®йўҳжүҖд»ҘиҮӘе·ұдё»иҰҒиҝҳжҳҜжҠҠASеҪ“еҒҡе°қйІңдёәдё»пјҢжҜҸж”ҫеҮәдёҖдёӘиҫғеӨ§жӣҙж–°е°ұдёӢиҪҪдёӢжқҘиҜ•иҜ•пјҢж„ҹи§үиҝҳжҳҜжҢәеҘҪзҡ„пјҢжёҗжёҗз”ЁASзҡ„дәәи¶ҠжқҘи¶ҠеӨҡпјҢGithubдёҠзҡ„йЎ№зӣ®д№ҹеҹәжң¬жҳҜASзҡ„дәҶпјҢGoogleзҡ„sampleд№ҹйҮҮз”ЁASпјҢжүҖд»ҘдҪҝз”ЁEclipseи·ҹеӨ–з•ҢдәӨжөҒи¶ҠжқҘи¶Ҡеӣ°йҡҫе•ҠгҖӮеҲ°зҺ°еңЁandroid-studio-bundle-135.1629389ASжӯЈејҸзүҲзҡ„жҺЁеҮәпјҢжҲ‘д»¬жңүзҗҶз”ұд»ҺEclipseиҝҒз§»еҲ°ASдәҶгҖӮ
иҰҒиҝҒз§»еҲ°ASдёӯејҖеҸ‘йӮЈд№ҲиҰҒжҺҢжҸЎASдёӯзҡ„йЎ№зӣ®з®ЎзҗҶжҳҜеҝ…йЎ»зҡ„пјҢеҹәжң¬зҡ„new projectгҖҒrunд»Җд№Ҳзҡ„е°ұдёҚжҸҗдәҶпјҢиҝҷзҜҮж–Үз« и®°еҪ•жҲ‘еңЁASдёӯеңЁйЎ№зӣ®дёӯи§ЈеҶіjarеҢ…гҖҒlibraryйЎ№зӣ®дҫқиө–гҖҒsoеҠЁжҖҒй“ҫжҺҘеә“зҡ„й—®йўҳпјҢзүҲжң¬жҺ§еҲ¶GitгҖҒSVNзӯүиҝҷзҜҮж–Үз« дёҚж¶үеҸҠгҖӮиҝҷд№ҲеӨҡеәҹиҜқеҗҺдёӢйқўејҖе§ӢгҖӮ

Eclipseи·ҹASзҡ„дёҚеҗҢ

д»ҺEclipseеҲ°ASдёҚиҰҒеёҰзқҖеңЁEclipseдёӯзҡ„дё»и§ӮиүІеҪ©еҺ»еңЁASдёӯдҪҝз”ЁпјҢд»ҺйЎ№зӣ®зҡ„жһ„жҲҗеҲ°жһ„е»әжҳҜдёҚеҗҢзҡ„пјҢдёӢйқўеҲ—дёҫеңЁEclipseе’ҢASдёӯзҡ„дёҖдәӣжҰӮеҝөзҡ„еҢәеҲ«пјҡ

WorkSpaceе’ҢProject

Eclipseзҡ„WorkSpaceе’ҢASзҡ„ProjectиҜҙзҡ„еҸҜд»ҘиҜҙжҳҜдёҖдёӘдёңиҘҝпјҢд№ҹе°ұжҳҜиҜҙдҪ еҸҜд»ҘжҠҠеңЁASдёӯзҡ„ProjectзҗҶи§ЈдёәWorkSpaceгҖӮ

жүҖд»ҘдҪ еңЁASдёӯnewдёҖдёӘProjectзӣёеҪ“дәҺеңЁEclipseдёӯйҮҚејҖдәҶдёҖдёӘWorkSpaceпјҢжіЁж„Ҹз¬¬дёҖдёӘз®ӯеӨҙпјҢжҳҫзӨәжЁЎејҸдёәProjectпјҢе»әи®®еҲҡз”ЁASж—¶з”Ёиҝҷз§ҚпјҢж–№дҫҝдәҶи§ЈйҮҢйқўзҡ„ж–Үд»¶з»“жһ„гҖӮ

Projectе’ҢModule

и·ҹдёҠйқўдёҖж ·пјҢEclipseдёӯзҡ„дёҖдёӘдёӘprojectд№ҹе°ұжҳҜзӣёеҪ“дәҺASдёӯзҡ„дёҖдёӘдёӘmoduleгҖӮдёҠеӣҫзҡ„module_1е’Ңmodule_2е°ұжҳҜжҲ‘д»¬д№ жғҜзҡ„eclipseдёӯзҡ„дёҖе ҶprojectдәҶпјҢжҠҠжҳҫзӨәжЁЎејҸжҚўдёәAndroidд№ӢеҗҺе°ұжӣҙдёәзӣҙи§ӮдәҶпјҡ

жңҖдёӢйқўзҡ„е°ұжҳҜASдёӯж•ҙдёӘProjectдёӯжүҖжңүGradleзҡ„й…ҚзҪ®дәҶпјҢеҪ“з„¶еҢ…жӢ¬жүҖжңүmoduleзҡ„й…ҚзҪ®дәҶпјҢжӢ¬еҸ·зҡ„еҗҚеӯ—е°ұиЎЁзӨәbuild.gradleеҜ№еә”зҡ„й…ҚзҪ®еҜ№иұЎгҖӮ

Propertiesе’ҢModule Setting

Eclipseдёӯзҡ„Propertiesд№ҹжҳҜи·ҹASзҡ„Module SettingеҜ№еә”зҡ„


еҸҜд»ҘзңӢеҲ°иҝҷйҮҢеҸҜд»ҘеғҸеңЁEclipseзҡ„PropertiesдёӯдёҖж ·еңЁиҝҷйҮҢй…ҚзҪ®дёҖдәӣдёңиҘҝпјҢжҜ”еҰӮеңЁModule SettingйҮҢз»ҷModuleж·»еҠ дҫқиө–(dependencies)дҝЎжҒҜд№ҹжҳҜеҸҜд»Ҙзҡ„пјҢе№¶дё”еҸҜд»ҘзӣҙжҺҘжҗңmavenзҡ„йЎ№зӣ®дҫқиө–гҖӮ

jar

жҳҺзҷҪдәҶдёҠйқўзҡ„дёүзӮ№е°ұеҸҜд»ҘеҫҲеҝ«дёҠжүӢдәҶгҖӮйҰ–е…Ҳе°ұжқҘиҜҙжңҖз®ҖеҚ•зҡ„ж·»еҠ jarеҢ…гҖӮ

  • еҸҜд»Ҙи·ҹеңЁEclipseдёӯдёҖж ·пјҢжҠҠjarеҢ…еҫҖModuleйҮҢжү”пјҢеҶҚеңЁjarеҸій”®add as libraryе°ұеҸҜд»ҘдәҶпјҢз„¶еҗҺжңҖеҗҺеңЁдҪ зҡ„Moduleж–Үд»¶еӨ№пјҲеғҸдёҠйқўзҡ„module_1пјүеҸій”®make moduleдёҖдёӢе°ұеҸҜд»ҘеңЁд»Јз ҒйҮҢз”ЁjarйҮҢзҡ„дёңиҘҝдәҶгҖӮ
  • д№ҹеҸҜд»ҘиҮӘе·ұжүӢеҠЁеҲ°moduleйҮҢзҡ„build.gradleйҮҢж·»еҠ dependenciesпјҢдёҠйқўзҡ„ж–№жі•еҒҡзҡ„ж–№жі•жң¬иҙЁдёҠе°ұжҳҜиҝҷз§ҚгҖӮ

dirеҸҜд»ҘжҳҜз”өи„‘дёҠзҡ„зӣ®еҪ•ж–Үд»¶гҖӮ

libraryйЎ№зӣ®

жңүдәҶеүҚйқўи·ҹEclipseзҡ„жҜ”иҫғеҗҺпјҢзұ»дјјең°еғҸеңЁEclipseдёӯж·»еҠ йЎ№зӣ®дҫқиө–дёҖж ·пјҢиў«дҫқиө–зҡ„йЎ№зӣ®еҫ—жҳҜдҪңдёәlibraryгҖӮеңЁEclipseдёӯжҲ‘д»¬жҳҜиҝӣе…ҘеҲ°PropertiesжҠҠиҝҷдёӘйЎ№зӣ®и®ҫзҪ®дёәlibraryпјҲas a libraryпјүпјҢжүҖд»ҘеңЁASдёӯд№ҹжҳҜзұ»дјјзҡ„пјҢжҲ‘д»¬йңҖиҰҒжҠҠдёҖдёӘmoduleдҪңдёәlibrary(иҝҷдёӘmoduleеҸҜд»ҘиҮӘе·ұж–°е»әmoduleд№ҹеҸҜд»ҘеҜје…ҘmoduleпјҢжӯӨеӨ–жҲ‘д»¬жҳҜеҸҜд»ҘжҠҠдёҖдёӘASзҡ„ProjectеҜјиҝӣжҲҗmoduleзҡ„жҲ–иҖ…зӣҙжҺҘеҜјProjectйҮҢзҡ„еҚ•дёӘmoduleд№ҹеҸҜд»Ҙ)пјҢе®ҢжҲҗеҗҺеҲ°иҝҷдёӘmoduleпјҲжҲ‘иҝҷйҮҢжҳҜжҠҠmodule_2дҪңдёәlibraryпјүжҠҠapply plugin: 'com.android.application'ж”№дёәapply plugin: 'com.android.library'еҶҚз„¶еҗҺеҺ»жҺүпјҲеҲ йҷӨпјүmodule_2зҡ„build.gradleйҮҢзҡ„applicationId "com.example.mrzheng.as"(дёҖдёӘlibraryдёҚйңҖиҰҒиҝҷдёӘпјҢдёҚз„¶make projectжҲ–make moduleж—¶дјҡжҠҘй”ҷ)гҖӮ
build.gradle(module_2)

еҲ°иҝҷйҮҢе…ҲзЎ®и®ӨдёӢдҪ зҡ„project(AS)зҡ„settings.gradleйҮҢжңүжІЎжҠҠmoduleйғҪincludeиҝӣеҺ»,жІЎжңүзҡ„иҜқеҠ дёҠпјҡ

жңҖеҗҺе°ұеҸҜд»ҘеңЁmodule_1йҮҢе°ұж·»еҠ libraryдҫқиө–(module_2)дәҶгҖӮиҝӣе…Ҙmodule_1зҡ„build.gradleпјҢжүҫеҲ°dependenciesеҠ дёҠcompile project(':module_2')
build.gradle(module_1)

зҺ°еңЁmake moduleдёҖдёӢе°ұеҸҜд»ҘдҪҝз”Ёдҫқиө–зҡ„йЎ№зӣ®дәҶ(module_2)гҖӮ

so

д№ӢеүҚзҡ„зүҲжң¬дёҚзҹҘйҒ“жҖҺд№Ҳж ·пјҢзҺ°еңЁжӯЈејҸзүҲзҡ„ASж·»еҠ soжү“еҢ…иҝӣapkйҮҢзҡ„libйҮҢжҳҜеҫҲз®ҖеҚ•зҡ„пјҢжҲ‘д»¬еҸӘйңҖиҰҒжҠҠsoж–Үд»¶ж”ҫеҲ°libsж–Үд»¶еӨ№йҮҢзҡ„еҜ№еә”cpuж–Үд»¶еӨ№йҮҢпјҢжңҖеҗҺеңЁmoduleзҡ„build.gradleйҮҢеҠ дёҠjniзҡ„sourceSetsй…ҚзҪ®пјҡjniLibs.srcDirs = ['libs']пјҢе®Ңж•ҙд»Јз ҒзңӢдёҠйқўзҡ„build.gradle(module_1)д»Јз ҒзүҮгҖӮ

END

еҒҡе®ҢдёҠйқўзҡ„е·ҘдҪңпјҢjarеҢ…гҖҒsoеҠЁжҖҒй“ҫжҺҘеә“гҖҒlibraryдҫқиө–йЎ№зӣ®йҮҢзҡ„д»Јз Ғе°ұйғҪеҸҜд»ҘжӯЈеёёе·ҘдҪңдәҶпјҢжӯЈеёёзҡ„йЎ№зӣ®ејҖеҸ‘е·Із»ҸжІЎд»Җд№Ҳй—®йўҳгҖӮжңҖеҗҺз»ҷдёҠжң¬еҚҡж–Үзҡ„жәҗз ҒпјҡAndroid Studio jarгҖҒsoгҖҒlibraryйЎ№зӣ®дҫқиө–пјҢеҜјиҝӣASйҮҢеҗҺпјҢrun module_1еҚіеҸҜгҖӮ
жңҖжңҖеҗҺеҘүдёҠAndroid Studioеҝ«жҚ·й”®еӨ§е…ЁпјҲеӨ§йғЁеҲҶжқҘжәҗдәҺпјҡhttp://ask.android-studio.org/?/article/12пјүпјҡ

йҷ„еҪ•пјҲIDEA(Android Studio) еҝ«жҚ·й”®пјү

иҜҙжҳҺпјҡж–ңдҪ“ж–Үеӯ—иЎЁзӨәпјҢжөӢиҜ•ж—¶жІЎжңүж•ҲжһңжҲ–иҖ…жІЎжңүжөӢиҜ•ж—¶жІЎжңүиҫҫеҲ°йў„е…ҲжқЎд»¶зҡ„жғ…еҶөдёӢжІЎжңүж•ҲжһңгҖӮ
