Androidдёӯзҡ„MVP

2015-02-06
  • Android

еүҚиЁҖ

MVPдҪңдёәдёҖз§ҚMVCзҡ„жј”еҢ–зүҲжң¬еңЁAndroidејҖеҸ‘дёӯеҸ—еҲ°дәҶи¶ҠжқҘи¶ҠеӨҡзҡ„е…іжіЁпјҢдҪҶеңЁйЎ№зӣ®ејҖеҸ‘дёӯйҖүжӢ©дёҖз§Қиҝҷж ·зҡ„иҪҜд»¶и®ҫи®ЎжЁЎејҸйңҖдҝқжҢҒж…ҺйҮҚеҝғжҖҒпјҢдёҖж—ҰзЎ®е®ҡдҪҝз”ЁMVPдҪңдёәдҪ Appзҡ„ејҖеҸ‘жЁЎејҸйӮЈд№ҲдҪ е°ұжңҖеҘҪеқҡжҢҒеҒҡдёӢеҺ»пјҢеҰӮжһңеңЁдҪҝз”ЁMVPжЁЎејҸејҖеҸ‘иҝҮзЁӢдёӯеҸ‘зҺ°й—®йўҳиҖҢдё”еқ‘и¶ҠжқҘи¶ҠеӨ§пјҢиҝҷж—¶дҪ жғіз”ЁMVCзӯүжқҘйҮҚж–°и®ҫи®Ўзҡ„иҜқеҹәжң¬дёҠе°ұзӯүдәҺжҺЁеҖ’йҮҚжқҘдәҶгҖӮиҰҒзҹҘйҒ“еңЁAndroidдёҠMVPеңЁзҺ°еңЁдёәжӯўе№¶жІЎжңүз»ҹдёҖзҡ„ж ҮеҮҶжҲ–иҖ…жЎҶжһ¶пјҢдёҚеғҸSSHиҝҷдёүдёӘжҲҗзҶҹзЁійҮҚејәиҖҢжңүеҠӣзҡ„дёүеү‘е®ўж”ҜжҢҒжҺЁеҠЁзқҖJava EEзҡ„ејҖеҸ‘пјҢжүҖд»ҘеңЁиҝҗз”ЁMVPж—¶дёҖе®ҡиҰҒеҒҡеҘҪиҮӘе·ұзҡ„зҗҶи§ЈпјҢе№¶дё”е°ҪйҮҸйў„зҹҘиҮӘе·ұAppеҗ„жЁЎеқ—зҡ„йңҖжұӮпјҲе®ўжҲ·иҜҙж”№ж”№ж”№пјҢжҲ‘д»¬е°ұж”№ж”№ж”№ :-( пјүд»ҘдҫҝжҸҗеүҚеҒҡеҘҪе……еҲҶзҡ„и®ҫи®Ўе·ҘдҪңгҖӮеҪ“з„¶MVPж—ўз„¶иғҪеҮәзҺ°йӮЈд№Ҳеҝ…з„¶жңүе®ғзҡ„дјҳзӮ№зҡ„пјҢдёҚз„¶и°ҒдјҡзҗҶдјҡиҝҷдёӘеҶ’еҮәжқҘзҡ„дёңиҘҝпјҢдёӢйқўе°ұеҜ№AndroidдёӯMVPеҒҡдёҖдәӣйҳҗиҝ°гҖӮ

MVPз®Җд»Ӣ

зӣёдҝЎеӨ§е®¶еҜ№MVCйғҪжҳҜжҜ”иҫғзҶҹжӮүдәҶпјҡM-Model-жЁЎеһӢгҖҒV-View-и§ҶеӣҫгҖҒC-Controller-жҺ§еҲ¶еҷЁпјҢMVPдҪңдёәMVCзҡ„жј”еҢ–зүҲжң¬пјҢд№ҹжҳҜдҪңдёәз”ЁжҲ·з•ҢйқўпјҲз”ЁжҲ·еұӮпјүзҡ„е®һзҺ°жЁЎејҸпјҢйӮЈд№Ҳзұ»дјјзҡ„MVPжүҖеҜ№еә”зҡ„ж„Ҹд№үпјҡM-Model-жЁЎеһӢгҖҒV-View-и§ҶеӣҫгҖҒP-Presenter-иЎЁзӨәеҷЁгҖӮд»ҺMVCе’ҢMVPдёӨиҖ…з»“еҗҲжқҘзңӢпјҢControlller/PresenterеңЁMVC/MVPдёӯйғҪиө·зқҖйҖ»иҫ‘жҺ§еҲ¶еӨ„зҗҶзҡ„и§’иүІпјҢиө·зқҖжҺ§еҲ¶еҗ„дёҡеҠЎжөҒзЁӢзҡ„дҪңз”ЁгҖӮиҖҢMVPдёҺMVCжңҖдёҚеҗҢзҡ„дёҖзӮ№жҳҜMдёҺVжҳҜдёҚзӣҙжҺҘе…іиҒ”зҡ„д№ҹжҳҜе°ұModelдёҺViewдёҚеӯҳеңЁзӣҙжҺҘе…ізі»пјҢиҝҷдёӨиҖ…д№Ӣй—ҙй—ҙйҡ”зқҖзҡ„жҳҜPresenterеұӮпјҢе…¶иҙҹиҙЈи°ғжҺ§ViewдёҺModelд№Ӣй—ҙзҡ„й—ҙжҺҘдәӨдә’пјҢMVPзҡ„з»“жһ„еӣҫеҰӮдёӢжүҖзӨәпјҢеҜ№дәҺиҝҷдёӘеӣҫзҗҶи§ЈеҚіеҸҜиҖҢдёҚеҝ…йҷҗдәҺе…¶дёӯзҡ„жқЎжқЎжЎҶжЎҶпјҢжҜ•з«ҹеңЁдёҚеҗҢзҡ„еңәжҷҜдёӢеӨҡе°‘дјҡжңүдәӣеҮәе…Ҙзҡ„гҖӮеңЁAndroidдёӯеҫҲйҮҚиҰҒзҡ„дёҖзӮ№е°ұжҳҜеҜ№UIзҡ„ж“ҚдҪңеҹәжң¬дёҠйңҖиҰҒејӮжӯҘиҝӣиЎҢд№ҹе°ұжҳҜеңЁMainThreadдёӯжүҚиғҪж“ҚдҪңUIпјҢжүҖд»ҘеҜ№ViewдёҺModelзҡ„еҲҮж–ӯеҲҶзҰ»жҳҜеҗҲзҗҶзҡ„гҖӮжӯӨеӨ–PresenterдёҺViewгҖҒModelзҡ„дәӨдә’дҪҝз”ЁжҺҘеҸЈе®ҡд№үдәӨдә’ж“ҚдҪңеҸҜд»ҘиҝӣдёҖжӯҘиҫҫеҲ°жқҫиҖҰеҗҲд№ҹеҸҜд»ҘйҖҡиҝҮжҺҘеҸЈжӣҙеҠ ж–№дҫҝең°иҝӣиЎҢеҚ•е…ғжөӢиҜ•гҖӮ

MVPд№ӢModel

Model жҳҜз”ЁжҲ·з•ҢйқўйңҖиҰҒжҳҫзӨәж•°жҚ®зҡ„жҠҪиұЎпјҢд№ҹеҸҜд»ҘзҗҶи§Јдёәд»ҺдёҡеҠЎж•°жҚ®пјҲз»“жһңпјүйӮЈйҮҢеҲ°з”ЁжҲ·з•Ңйқўзҡ„жҠҪиұЎпјҲBusiness rule, data access, model classesпјүгҖӮжң¬ж–Ү Demo дёәдәҶз®ҖеҚ•еӨ„зҗҶе°ұзӣҙжҺҘжҠҠдёҡеҠЎж”ҫеҲ°дәҶеҜ№еә”зҡ„ Model д№ӢдёӯгҖӮ

MVPд№ӢView

и§ҶеӣҫиҝҷдёҖеұӮдҪ“зҺ°зҡ„еҫҲиҪ»и–„пјҢиҙҹиҙЈжҳҫзӨәж•°жҚ®гҖҒжҸҗдҫӣеҸӢеҘҪз•Ңйқўи·ҹз”ЁжҲ·дәӨдә’е°ұиЎҢгҖӮMVPдёӢActivityе’ҢFragmentдҪ“зҺ°еңЁдәҶиҝҷдёҖеұӮпјҢActivityдёҖиҲ¬д№ҹе°ұеҒҡеҠ иҪҪUIи§ҶеӣҫгҖҒи®ҫзҪ®зӣ‘еҗ¬еҶҚдәӨз”ұPresenterеӨ„зҗҶзҡ„дёҖдәӣе·ҘдҪңпјҢжүҖд»Ҙд№ҹе°ұйңҖиҰҒжҢҒжңүзӣёеә”Presenterзҡ„еј•з”ЁгҖӮдҫӢеҰӮпјҢActivityдёҠж»ҡеҠЁеҲ—иЎЁж—¶йҡҗи—ҸжҲ–иҖ…жҳҫзӨәAcionbarпјҲToolbarпјүпјҢиҝҷж ·зҡ„UIйҖ»иҫ‘ж—¶д№ҹеә”иҜҘеңЁиҝҷдёҖеұӮгҖӮеҸҰеӨ–еңЁViewдёҠиҫ“е…Ҙзҡ„ж•°жҚ®еҒҡдёҖдәӣеҲӨж–ӯж—¶пјҢдҫӢеҰӮпјҢEditTextзҡ„иҫ“е…Ҙж•°жҚ®пјҢеҒҮеҰӮжҳҜз®ҖеҚ•зҡ„йқһз©әеҲӨж–ӯеҲҷеҸҜд»ҘдҪңдёәViewеұӮзҡ„йҖ»иҫ‘пјҢиҖҢеҪ“йңҖиҰҒеҜ№EditTextзҡ„ж•°жҚ®иҝӣиЎҢжӣҙеӨҚжқӮзҡ„жҜ”иҫғж—¶пјҢеҰӮд»Һж•°жҚ®еә“иҺ·еҸ–жң¬ең°ж•°жҚ®иҝӣиЎҢеҲӨж–ӯж—¶жҳҺжҳҫйңҖиҰҒз»ҸиҝҮModelеұӮжүҚиғҪиҝ”еӣһдәҶпјҢжүҖд»Ҙиҝҷдәӣз»ҶиҠӮйңҖиҰҒиҮӘе·ұжҺӮйҮҸгҖӮ

MVPд№ӢPresenter

PresenterиҝҷдёҖеұӮеӨ„зҗҶзқҖзЁӢеәҸеҗ„з§ҚйҖ»иҫ‘зҡ„еҲҶеҸ‘пјҢж”¶еҲ°ViewеұӮUIдёҠзҡ„еҸҚйҰҲе‘Ҫд»ӨгҖҒе®ҡж—¶е‘Ҫд»ӨгҖҒзі»з»ҹе‘Ҫд»ӨзӯүжҢҮд»ӨеҗҺеҲҶеҸ‘еӨ„зҗҶйҖ»иҫ‘дәӨз”ұдёҡеҠЎеұӮеҒҡе…·дҪ“зҡ„дёҡеҠЎж“ҚдҪңпјҢз„¶еҗҺе°Ҷеҫ—еҲ°зҡ„ Model з»ҷ View жҳҫзӨәгҖӮ

жј”зӨәdemo

еҠЁжүӢеҶҷиө·д»Јз ҒжқҘжүҚжңүжӣҙеҘҪзҡ„ж„ҹи§үгҖӮdemoеҫҲз®ҖеҚ•,иҝҳжҳҜдёҠдёӘеӣҫжӣҙзӣҙи§ӮпјҢиҫ“е…ҘеҹҺеёӮзҡ„д»ЈеҸ·пјҢзӮ№еҮ»жҢүй’®иҺ·еҸ–еҹҺеёӮзҡ„еӨ©ж°”дҝЎжҒҜз„¶еҗҺжҳҫзӨәеҮәжқҘпјҢзҪ‘з»ңж“ҚдҪңдҪҝз”ЁVolleyжЎҶжһ¶пјҢи§Јжһҗз”ЁGsonпјҢе…¶е®ғзҡ„е°ұжүӢеҶҷдәҶгҖӮж•ҙдёӘйЎ№зӣ®зҡ„еҢ…и®ҫи®ЎеҰӮдёӢпјҡ



еҢ…еӣҫдёӯжҳҺжҳҫзҡ„дёүеұӮпјҡModelеҢ…гҖҒPresenterеҢ…гҖҒUIеҢ…пјҢе…¶дёӯпјҢдёүиҖ…йғҪе®һзҺ°еҗ„иҮӘзҡ„з»“жһ„пјҢModelдёәWeatherModelгҖҒPresenterдёәWeatherPresenterгҖҒViewдёәWeatherпјҢйӮЈд№Ҳе…·дҪ“е®һзҺ°зұ»е°ұжҳҜimplеҢ…йҮҢзҡ„дәҶпјҢViewеұӮзҡ„еҚідёәActivityгҖӮжӯӨеӨ–зҡ„appе’ҢutilеҢ…ж— е…ізҙ§иҰҒеҸҜд»ҘдёҚзңӢгҖӮеҸҜд»ҘзңӢеҲ°йҮҮз”ЁMVPи®ҫи®ЎеҗҺйЎ№зӣ®жҳҺжҳҫеӨҡдәҶеҫҲеӨҡдёңиҘҝпјҢиҝҷд№ҹжҳҜдёҚеҸҜйҒҝе…Қзҡ„пјҢдҪҝз”ЁеҺҹе§Ӣж–№жі•еҸҜд»ҘдҪҝйЎ№зӣ®ејҖиө·жқҘз®ҖеҚ•дәӣдҪҶжҳҜд»ҘеҗҺиҝҳжңүз»ҙжҠӨе‘ўгҖҒжөӢиҜ•е‘ўгҖҒеҠ еҠҹиғҪе‘ўгҖҒгҖӮгҖӮгҖӮ
entityйҮҢзҡ„е®һдҪ“еұһжҖ§еҹәжң¬дёҠеҜ№еә”jsonйҮҢзҡ„иҝҷдәӣеұһжҖ§дәҶпјҢд»Јз ҒдёҚиҙҙдәҶпјҢViewйҮҢйқўзҡ„жҺҘеҸЈпјҡ

WeatherPresenterзҡ„жҺҘеҸЈпјҡ

WeatherModelжҺҘеҸЈпјҡ

prestenerйҮҢйқўиҝҳжңүдёӘOnWeatherListenerпјҢе…¶еңЁPresenterеұӮе®һзҺ°пјҢз»ҷModelеұӮеӣһи°ғпјҢжӣҙж”№ViewеұӮзҡ„зҠ¶жҖҒпјҢзЎ®дҝқModelеұӮдёҚзӣҙжҺҘж“ҚдҪңViewеұӮгҖӮеҰӮжһңжІЎжңүиҝҷдёҖжҺҘеҸЈеңЁWeatherPresenterImplе®һзҺ°зҡ„иҜқпјҢWeatherPresenterImplеҸӘжңүViewе’ҢModelзҡ„еј•з”ЁйӮЈд№ҲModelжҖҺд№ҲжҠҠз»“жһңе‘ҠиҜүViewе‘ўпјҹеҪ“з„¶иҝҷеҸӘжҳҜдёҖз§Қи§ЈеҶіж–№жЎҲпјҢеңЁе®һйҷ…йЎ№зӣ®дёӯеҸҜд»ҘдҪҝз”ЁDaggerгҖҒEventBusгҖҒOttoзӯүз¬¬дёүж–№жЎҶжһ¶з»“еҗҲиҝӣжқҘиҫҫеҲ°жӣҙеҠ жқҫиҖҰеҗҲзҡ„и®ҫи®ЎгҖӮ

жүҖд»Ҙdemoзҡ„д»Јз ҒжөҒзЁӢпјҡActivityеҒҡдәҶдёҖдәӣUIеҲқе§ӢеҢ–зҡ„дёңиҘҝе№¶йңҖиҰҒе®һдҫӢеҢ–еҜ№еә”WeatherPresenterзҡ„еј•з”Ёе’Ңе®һзҺ°WeatherViewзҡ„жҺҘеҸЈпјҢзӣ‘еҗ¬з•ҢйқўеҠЁдҪңпјҢGoжҢүй’®жҢүдёӢеҗҺеҚіжҺҘж”¶еҲ°жҹҘиҜўеӨ©ж°”зҡ„дәӢд»¶пјҢеңЁonClickйҮҢжҺҘж”¶еҲ°еҚійҖҡиҝҮWeatherPresenterзҡ„еј•з”ЁжҠҠе®ғдәӨз»ҷWeatherPresenterеӨ„зҗҶгҖӮWeatherPresenterжҺҘж”¶еҲ°дәҶжҹҘиҜўеӨ©ж°”зҡ„йҖ»иҫ‘е°ұзҹҘйҒ“иҰҒжҹҘиҜўеӨ©ж°”дәҶпјҢз„¶еҗҺжҠҠжҹҘиҜўеӨ©ж°”зҡ„е…·дҪ“дёҡеҠЎе®һзҺ°дәӨз»ҷWeatherModelеҺ»е®һзҺ°еҗҢж—¶жҠҠWeatherListenerеҚіWeatherPresenterиҮӘе·ұдј з»ҷWeatherModelгҖӮWeatherModelиҝӣиЎҢжҹҘиҜўеӨ©ж°”дёҡеҠЎеҗҺеҚіжҠҠз»“жһңйҖҡиҝҮWeatherListenerеӣһи°ғйҖҡзҹҘWeatherPresenterпјҢWeatherPresenterеҶҚжҠҠз»“жһңиҝ”еӣһз»ҷViewеұӮзҡ„ActivityпјҢжңҖеҗҺActivityжҳҫзӨәз»“жһңгҖӮе°ұиҝҷж ·пјҢжӢҚз –д№ӢеӨ„иҜ·жӢҚгҖӮ
